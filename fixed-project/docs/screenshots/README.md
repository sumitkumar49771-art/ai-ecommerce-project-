# Screenshots

Drop your screenshots here with these exact names so they show up automatically
in the main project README:

- `home.png` — homepage
- `products.png` — product listing page
- `admin.png` — admin dashboard
- `mobile.png` — mobile view (use Chrome DevTools device toolbar, Ctrl+Shift+M)

Tip: use your OS screenshot tool (Win + Shift + S on Windows) to capture just
the page content — not the whole browser tab bar — so it looks clean.
